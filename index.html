One cat, three tools

Each piece works on its own, and all three talk to each other when you want them to.

Desktop App

Download video and audio with the app under the hood, browse a Stremio-compatible catalog, and watch it all in a built-in player.

See how it works →

Browser Extension

A lightweight companion for Chrome, Edge and Brave that hands any page's video straight to the desktop app.

See how it works →

MCP Server

Connect BuckyMedia to Claude Desktop and other MCP-compatible AI agents, so an agent can queue downloads for you.

See how it works →
DOWNLOADER

A complete video/audio downloader

The downloader lets you download videos, audio, subtitles and a specific audio track from hundreds of websites in the most popular formats and even in 8K and 4K resolutions. To use the advanced download features, simply enable "Advanced Mode" in the app settings.

Explore the downloader tour →
CATALOG

A movies-and-series catalog you actually control

The catalog speaks the same protocol as Stremio addons, so any compatible addon plugs right in. Install one with an API key, paste a manifest URL, and reorder your addons to decide whose results show up first.
